Project Overview

The Daily Expense Tracker is a comprehensive financial management tool designed to help users track their daily spending habits.


Built as a frontend-only application with local data storage, ensuring privacy and security.


Features a modern, responsive design that works seamlessly across all devices.

2 / 12

Key Features

  • User Authentication: Secure login and registration system
  • Multiple Users: Individual accounts with data separation
  • Expense Management: Add, view, and delete expenses
  • Categorization: 6 predefined expense categories
  • Real-time Summary: Total and daily expense tracking
  • Responsive Design: Works on all devices
  • Data Persistence: Local storage with session management
